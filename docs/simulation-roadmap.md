# 物理仿真开发清单

目标：平台按教材、章节、知识点组织。左侧是课程结构，右侧是可调参数的可视化仿真。仿真优先服务教学理解，不追求游戏化装饰。

## 已完成方向

- 初中三册教材结构已录入。
- 高中必修与选择性必修的主要章节骨架已录入。
- 知识点下面直接挂对应模型。
- 右侧主区域改为大仿真画布。
- 已有基础参数和观测量面板。

## 需要继续细化的模型

- 运动学：补自由落体、追及相遇、打点计时器数据拟合。
- 力学：补胡克定律、摩擦力临界、牛顿第二定律真实运动。
- 曲线运动：补平抛落点、斜抛最大高度、圆周临界条件。
- 静电场：补可拖动电荷、等势线、带电粒子加速和偏转。
- 电路：补可拖动滑动变阻器、串并联切换、电表量程。
- 磁场与电磁感应：补洛伦兹力轨迹、线圈转动、交流电波形。
- 光学：补凸透镜物距/像距可调、全反射临界角。
- 热学：补扩散粒子随机运动、比热容加热曲线。

## 暂缓决策

- 是否引入真实物理引擎，例如 Matter.js、Planck.js 或自研数值积分。
- 是否用 Canvas/WebGL 替换部分 SVG 仿真。
- 是否做教师端编辑器，让用户自己配置模型参数、题目和讲解词。
- 是否接入视频渲染队列，把当前仿真状态直接导出成教学视频。
