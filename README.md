# Physics Visual Platform

一个用于初高中物理题目可视化视频生产的本地 Web 平台。

目标风格：极简、白底、几何图形、清晰受力箭头、分步动画，适合把课件或题目转成类似“物理题可视化”的讲解视频。

## 当前能力

- 初高中物理知识点库雏形
- 常见模型模板库
  - 斜面滑块/木板
  - 电路动态分析
  - 宏观到微观放大
  - 运动图像
  - 滑轮组
  - 带电粒子场中运动
- 题目工作台
- 自动生成分镜草案
- SVG 极简可视化画布

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 推荐生产流程

1. 输入题目或课件内容。
2. 识别知识点、物理对象、约束条件和目标量。
3. 匹配模型模板。
4. 生成分镜、字幕、配音稿和动画参数。
5. 用 HyperFrames 或 Remotion 渲染 MP4。
